# Budget-Microcentrifuge
A lab-grade microcentrifuge designed to prove the viability of low-cost DIY lab equipment. Safely reaches speeds of up to 3000 RPM.

Takes 1.5ml Eppendorf Tubes

Components:

12V-40v DC Motor Controller	£8

DC 775 Motor 12V/24V (D-shaft)	£16

12v 5A 60W LED Power Supply AC-DC	£14

(10) M3 Screws

Total Cost	£38

All custom parts are included in the repository. All parts can be 3D-printed except for the acrylic lid, which should be laser-cut or CNC'd.
